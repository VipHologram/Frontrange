Admin Setup (One Time Only)

Create your private cloud database. After this, all devices will sync automatically.

🔐 Create GitHub Token (gist scope only):
1. Go to GitHub Settings → Tokens
2. Click "Generate new token (classic)"
3. Select ONLY gist scope
4. Copy the token (starts with ghp_)

⚠️ Keep this token safe. All devices will use it to sync data.
